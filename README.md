# 🎮 Tic Tac Toe

A classic Tic Tac Toe game built with React.

## 🚀 Live Demo
👉 [https://tic-tac-toe-harshitha.vercel.app](https://tic-tac-toe-harshitha.vercel.app)



## ✨ Features

- Two-player gameplay (Player X & Player O)
- Score tracking for both players
- Win detection for rows, columns, and diagonals
- Draw detection
- Reset game functionality

## 🛠️ Tech Stack

- **React** — UI library
- **CSS** — Styling
- **Vercel** — Deployment

## 🚀 Getting Started

```bash
git clone https://github.com/sriharshitha-konkathi/tic-tac-toe.git
cd tic-tac-toe
npm install
npm start
```
